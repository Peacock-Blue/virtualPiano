# Virtual Piano
This project tries to simulate a Piano, making use of Augmented Reality and Computer Vision.

Prerequisites: <br/>
 * Python3.8
 * Python modules:
   - gensound
   - mediapipe
   - OpenCV-2

How to use?
1. Clone the repo
2. `cd <repo_name>`
3. `python3 camera.py`
4. Do you see the Piano? You know what to do with it!
